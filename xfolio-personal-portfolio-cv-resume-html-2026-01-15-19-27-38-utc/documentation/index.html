SCSS Files and Structure


Mainly three main CSS files are used in this theme. The first one bootstrap.css, second one is main.css which for this template and third one is responsive.css to control responsive layouts.

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

We used SCSS for better productivity


Instead of working with large css file we break them into three pieces

abstracts: We keep every mixins and variables in this folder. (Note: For naming convention we used BEM method.)

base: Global styles are written here.

components: All major components styles are written here.

layout: Layouts (i.e: header, footer etc) styles are written here.

sections: Section specific styles are written here.


Jquery and javascript


This theme imports two types of Javascript files.

  • jQuery
  • Use Some plugins file
  • Custom scripts (main.js)

  • jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  • Most of the animation in this site is carried out from the customs scripts. There are a few functions worth looking over.

In addition to the custom scripts, I have implemented few "tried and true" plugins to create the effects. This plugin is packed, so you won't need to manually edit anything in the file. The only necessary thing to know is how to call the method
